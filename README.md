## 鼠须管自制配色

自己没事用 [Rime 西米 for Squirrel](https://gjrobert.github.io/Rime-See-Me-squirrel/) 调制的 Mac 中文输入法鼠须管各种配色（color_scheme），适用于竖排，如下：

![github 暗色主题](img/github.png)

其他外观没有具体设置配色，所以可能会出现问题。

## 主题效果：

查看主题效果可前往 `img/` 文件夹查看。

## 使用方法：

1. 把 `squirrel.custom.yaml` 放到 Rime 根目录。

2. 第六行选择自己喜欢的主题。如，
```yaml
color_scheme: github 
```

3. 重新部署即可。
